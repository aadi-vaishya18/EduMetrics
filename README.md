**EduMetrics** is a modern, interactive Online Examination System built entirely with Vanilla JavaScript, HTML, and CSS. Designed with a premium, glassmorphic user interface, the platform delivers a strict, mock-secure assessment environment that looks beautiful and acts responsively across all devices. 

The application is structured into a dual-role architecture:
- **Admin Portal**: Allows educators to dynamically build customizable exams featuring polymorphic question formats—supporting Multiple Choice, True/False, and intelligent Keyword-matched Descriptive inputs. 
- **Student Portal**: Provides an isolated, streamlined iteration loop for secure test-taking.

*Note: The platform currently operates strictly as an in-memory client-side demonstration to practice robust frontend Data Structures and Object-Oriented Principles.*

## ✨ Key Features

- **Object-Oriented Programming (OOP) Architecture:** Utilizes strictly governed class systems and polymorphic evaluation logic natively in the browser.
- **Custom Stack Mechanism:** Powers a dynamic, resilient 'Undo Last Answer' workflow for test-takers.
- **Custom Queue Processing:** Efficiently handles background evaluations and concurrent submission pipelines.
- **Native Selection Sorting:** Utilizes hand-coded sorting algorithms to compute grades and generate instantaneous, ranked Merit Lists.
- **Glassmorphic UI:** A visually stunning layout offering dynamic feature cards and polished form interactions.
- **Dynamic Question Builder:** Build evaluations without rigid schema limitations using an intuitive admin interface.

## 🛠️ Technology Stack

- **Core:** HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Hosting:** Firebase Hosting
- **Architecture Features:** Custom Stacks, Queues, Algorithms, and OOP Principles

## ⚙️ Running Locally

Because this project is built entirely strictly with vanilla web technologies, you don't even need a bundler to run it!

1. Clone the repository:
   ```bash
   git clone https://github.com/aadi-vaishya18/EduMetrics.git
   ```
2. Open the directory:
   ```bash
   cd EduMetrics
   ```
3. Open `index.html` in your favorite web browser (or serve using an extension like *Live Server* in VSCode).

## 💡 About The Code

The codebase is organized modularly for logical separation:
- `app.js` and `app.html` handle all testing infrastructure and session runtime logic.
- `index.html` and `landing.css` form a lightweight, premium routing entry-point into the core application.
- `.firebaserc` and `firebase.json` are maintained for frictionless deployments to Firebase.
